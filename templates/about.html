About the Volunteer Coordination Platform

Our Mission

To empower communities by creating an efficient, secure, and user-friendly platform that connects organizations with passionate volunteers, enabling meaningful volunteer experiences and community impact.

Features

  • Role-Based Access: Separate dashboards and features for organizations and volunteers
  • JWT Authentication: Secure login with tokens and HTTP-only cookies
  • Event Management: Create, edit, and manage volunteer events with full details
  • Smart Registration: Prevent duplicate registrations and time conflicts
  • Real-time Status: Automatic event status updates (Open/Full)
  • Responsive Design: Works on desktop, tablet, and mobile devices
  • Data Validation: Comprehensive input validation for security

Technology Stack

Backend: Python Flask with Flask-SQLAlchemy ORM

Frontend: HTML5, CSS3, Vanilla JavaScript with Jinja2 templating

Database: MySQL

Authentication: JWT with Flask-JWT-Extended

Security: Werkzeug password hashing

How It Works

For Volunteers:

  1. Sign up with your name, email, phone, and password
  2. Browse all available volunteer events
  3. Register for events that interest you
  4. View all your registrations in one place
  5. Unregister from events if plans change

For Organizations:

  1. Register your organization with details and description
  2. Create new volunteer events with time, location, and volunteer requirements
  3. Track registrations in real-time
  4. Edit or delete events as needed
  5. View your event dashboard with all key metrics
